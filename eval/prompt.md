# Test Prompt

Use this exact prompt for both runs (with and without skill-chain).

---

Build a single-page landing page for Browser Use (browser-use.com) — an open-source tool that lets AI agents control web browsers. The page must fit in a single viewport with no scrolling.

Requirements:
- Hero section with headline, subheadline, and primary CTA
- Key value props (3-4 max) shown visually, not as a text wall
- Social proof element (GitHub stars, contributor count, or similar)
- Clean, modern design that feels like a real product — not AI-generated
- Next.js + Tailwind CSS
- Dark theme
- Must render fully above the fold on a 1440x900 viewport

Do not use placeholder images. Use SVG graphics or CSS-only visual elements.
